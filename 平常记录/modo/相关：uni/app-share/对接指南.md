## 一、接入准备

1. 拷贝 `nativeplugins/Modo-AppShare` 到工程
2. `manifest.json` → **App 原生插件配置** → 勾选 **Modo-AppShare**
3. 拷贝 iOS SDK（首次或升级时）：
   ```bash
   ./scripts/run-all-copy-appshare-sdk.sh
   ```
4. 制作 **iOS 自定义调试基座**（修改 `nativeplugins` 内任意 `.m` 后必做）：
   ```bash
   ./scripts/prepare-offline-hello-archive.sh
   # Xcode Archive → 导出 ipa →
   ./scripts/install-ios-debug-ipa.sh /path/to/HBuilder.ipa
   ```
5. HBuilderX 运行到 iOS 时勾选 **使用自定义基座**

仅改 `.vue` / `js_sdk` **不必**重打基座。

## 二、引入与监听

```javascript
import {
  shareImage,
  onShareEvent,
  offShareEvent,
  probeShareEnvironment,
  isPluginAvailable
} from '@/js_sdk/app-share/index.js'

onShareEvent((evt) => {
  console.log(evt.status, evt.code, evt.message, evt.platform)
})

shareImage({
  platform: 'wx',
  contentType: '1',
  opentype: '1',
  wechatAppId: 'wxYourAppId',
  wechatUniversalLink: 'https://your.domain/path/',
  shareContent: imageUrl,
  shareTitle: '标题',
  shareDes: '描述'
})
```

`App.vue` 已在 `onShow` / `newintent` 转发 URL 与 Universal Link（微信 / QQ / 支付宝 / 抖音回跳）。

## 七、抖音注意

1. **不要**在 `applicationDidFinishLaunching` 里注册抖音；用 `opentype: '0'` 或分享前自动注册。
2. 自定义基座若 `opentype: '2'` 闪退，先用 **`opentype: '8'`** 系统分享验证链路。
3. 相册选图分享需 **iOS 14+**；SDK 分享使用相册 `localIdentifier`。
4. 离线 Hello 需 Embed `DouyinOpenSDK`、`DYOpenCore.bundle`（`prepare-offline-hello-archive.sh` 已处理）。
