# 控制组件：测试用例

## 1. 前置条件

- 真机 **iOS 18+**。
- 主 App 与 **HYControlWidgetExtension** 同一 App Group（`NTControlCenterAppGroupSuite` 或推导 `group.<主包ID>`）。
- Pod **ControlWidgetCore** 主 App + 扩展均已链入。
- 宿主 `HYControlWidgetOpenApp.swift` **双 Target**；Bundle 使用宿主 `ControlWidget`。
- NativeBility `control` 插件已注册；游戏可调用 `query` / `getInfo`。
- 主 App `CFBundleURLTypes` 与 `HYControlWidgetConfig.openURL` 一致。

---

## 2. 控制入口与跳转

| ID | 操作 | 预期 |
|----|------|------|
| C-01 | 控制中心添加「花园·jump」 | 图标、文案正确 |
| C-02 | App 后台点击 jump | 主 App 前台/拉起 |
| C-03 | 杀进程后点击 | 冷启动 |
| C-04 | 点击 jumpWrite | 同样拉起；`getInfo` 为 `jumpWrite` |
| C-05 | iOS 17 及以下 | 无控制中心控件或不可用，主 App 不崩 |

---

## 3. App Group 与 TTL

| ID | 操作 | 预期 |
|----|------|------|
| A-01 | 点击 writeToGroup=true 的按钮 | `ControlWidget` 下出现对应 `intentName` 子键 |
| A-02 | 主 App `query` 用 `{}` | `fromControlCenter=true`，`matchedIntentName` 正确；磁盘子键被删 |
| A-03 | 再次 `query` | `fromControlCenter=false`（已消费） |
| A-04 | 构造 `ts` 超过 30s 前 | `query` false；日志提示过期；purge 清理 |
| A-05 | 扩展 suite 与主 App 不一致 | 写盘有、query 无；查 `[control]` 日志 suite |

---

## 4. NativeBility query / getInfo

| ID | 操作 | 预期 |
|----|------|------|
| N-01 | 点击后立即 `query` `{}` | `fromControlCenter=true` |
| N-02 | 同次启动接着 `getInfo` | 返回 `intentName`/`ts`/`isControlOpen` |
| N-03 | 仅 `getInfo` 未先 query | 仍可从盘读并消费（或空） |
| N-04 | `getInfo` 带 `intentName: jumpWrite` 过滤 | 仅匹配 jumpWrite |
| N-05 | 手动打开 App（未点控制） | `query` false |
| N-06 | `delegate` 清空后再 `query` | false |

调试注意：**不要**在测 `query` 时传错误 `intentName`，适配器调试数据应使用 **`{}`**。

---

## 5. 架构回归（防踩坑）

| ID | 操作 | 预期 |
|----|------|------|
| R-01 | Release 包点击控制 | 能跳主 App（wire + 双 Target Intent） |
| R-02 | Bundle 仅 `ControlWidgetCoreOpenAppControl` | **失败用例**：真机应不跳（勿发版） |
| R-03 | 改 kind 后不删控件 | 可能仍旧行为；删控件重加后正常 |
| R-04 | `AppIntent.title` 非字面量 | 编译失败（预期） |
| R-05 | URL scheme 与 plist 不一致 | 不跳或跳失败 |

---

## 6. 游戏业务（自动收花，若已接）

| ID | 操作 | 预期 |
|----|------|------|
| G-01 | `getInfo` 后分支 `jump` | 走打开/收花逻辑（按产品） |
| G-02 | 无成熟鲜花 | 仅进游戏 |
| G-03 | 控制拉起后、收花前切后台 | 取消本次自动收花（游戏态机） |
| G-04 | 收花请求已发出后切后台 | 不重复客户端请求；服务端幂等 |

---
