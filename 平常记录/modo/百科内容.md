3. 公钥秘钥
   ```
   一、密钥生成
    mac 系统如何生成秘钥的方法，记录在此：
   
   1. 打开终端，输入 cd ~/.ssh，检查是否已经存在了SSH密钥。如果你看到类似id_rsa.pub的文件，说明你已经有了一对公钥和私钥，可以跳过第 2 步和第 3 步。
   
   2. 在终端输入ssh-keygen -t rsa -C "你的邮箱地址" ，生成新的SSH密钥。你可以直接按回车键使用默认的文件路径和空密码，也可以自己设置。
   
   3. 在终端输入ssh-add ~/.ssh/id_rsa，添加SSH密钥到 ssh-agent。如果你之前设置了密码，这里需要再次输入。
   
   4. 在终端输入vim ~/.ssh/id_rsa.pub，查看并复制你的公钥内容。你可以把公钥添加到你需要远程访问的代码仓库网站或服务器上。
   
   5.复制
   pbcopy < ~/.ssh/id_rsa.pub
   ```

   ### 📂 目录与文件说明

   | 文件名            | 作用                                                         |
   | ----------------- | ------------------------------------------------------------ |
   | `id_rsa`          | 👉 这是 **SSH 私钥文件**，非常重要！只能你自己拥有，不能泄露给别人。它用来证明“你是你”。 |
   | `id_rsa.pub`      | 👉 这是 **SSH 公钥文件**，可以安全地提供给别人（比如 GitHub、GitLab、服务器管理员）。别人用它来验证你的身份。 |
   | `known_hosts`     | SSH 曾经连接过的服务器记录列表。系统会用它来验证是否连接到了可信的服务器（防止中间人攻击）。 |
   | `known_hosts.old` | 旧的 `known_hosts` 备份文件。                                |

   ### 简单来说：

   - `id_rsa` + `id_rsa.pub` = 你的一对 SSH 密钥。
   - 公钥放远端（比如 GitHub），私钥留在本地。
   - SSH 登录或 `git clone` 时用它来做无密码验证。
