共用一套bundle操作步骤：

1. 打开配置文件：

```
nano ~/.zshrc    # 如果你用 zsh
```

1. 在文件末尾加入：

```
# 全局 gem 安装路径
export BUNDLE_PATH="$HOME/.bundle"

# 全局 gem 缓存路径
export BUNDLE_CACHE_PATH="$HOME/.bundle/cache"

# 如果用 rbenv 或 chruby，确保 gem 环境也正常
```

1. 保存并退出（在 nano 里按 `Ctrl+O` 回车保存，`Ctrl+X` 退出）
2. 让配置生效：

```
source ~/.zshrc
```

1. 验证：

```
bundle config get path
bundle config get cache_path
```

- 应该显示：

```
/Users/你的用户名/.bundle
/Users/你的用户名/.bundle/cache
```
