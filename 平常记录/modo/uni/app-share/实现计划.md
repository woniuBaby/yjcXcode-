# Modo-AppShare 分步实现计划

> 参考：`uniapp_plugins_demo/uni_modules/app-share`  
> 原则：**一次只接一个平台，真机验证通过后再做下一个**；JS API 与 Demo 布局与 uni-x 保持一致。

---

## 总览

| 步骤 | 平台 | iOS 状态 | 验收 |
|------|------|----------|------|
| 0 | 抖音 douyin | ✅ | 系统分享 / 初始化 |
| **1** | **微信 wx** | **✅** | 聊天 + 朋友圈 + 回调 |
| **2** | **QQ qq** | **✅** | 好友 + QZone |

## 步骤 1：微信（已完成代码，待你 Archive 验证）

### 原生

- `ios/wx/ModoAppShareWxBridge.m` — 对齐 `AppShareWxAdapter.swift`
- `ios/ModoAppShareBridge.m` — 按 `platform` 路由
- SDK：`scripts/copy-wechat-sdk-to-appshare.sh` → `WechatOpenSDK-NoPay.xcframework`

### plist（Demo 测试 AppId）

- URL Scheme：`wx4268b80776894552`
- `LSApplicationQueriesSchemes`：`weixin` / `weixinULAPI` 等

### JS / Demo

- `shareImage({ platform:'wx', opentype:'1'|'2', wechatAppId, wechatUniversalLink, shareContent:imageUrl })`
- `pages/app-share/share-demo.vue` — 微信区块

### 验收清单

- [ ] 自定义基座含 WechatOpenSDK
- [ ] 分享到聊天成功，事件 `status=success`
- [ ] 分享到朋友圈成功
- [ ] 未装微信时 `wechat_not_install`
- [ ] 从微信返回 App，`handleOpenURL` 收到回调

## 步骤 2：QQ（已完成代码，待你 Archive 验证）

### 原生

- `ios/qq/ModoAppShareQQBridge.m` — 对齐 `AppShareQQAdapter.swift`
- SDK：`scripts/copy-qq-sdk-to-appshare.sh` → `TencentOpenAPI.framework`
- plist：URL Scheme `tencent101998280`、queries `mqq` 等（已在 `package.json`）

### Demo

- `share-demo.vue` — 「分享到 QQ 好友 (1)」「分享到 QQ 空间 (2)」

### 验收清单

- [ ] 自定义基座含 TencentOpenAPI
- [ ] 分享到 QQ 好友成功
- [ ] QZone 图片分享（opentype=2；若 10001 则改测好友）
- [ ] 从 QQ 返回 App 收到 `ModoAppShareEvent`

### API（与 uni-x 一致）

```javascript
shareImage({
  platform: 'qq',
  opentype: '1', // 好友；2=QZone
  tencentAppId: '101998280',
  tencentUniversalLink: 'https://...',
  shareContent: imageUrl,
  shareTitle, shareDes, imageUrl
})
```

### 验收清单

- [ ] 自定义基座含 libAPOpenSdk.a
- [ ] opentype=0 返回 `debug_init_ok`
- [ ] opentype=1 拉起支付宝分享链接卡片


## 基座流程

修改 `nativeplugins/Modo-AppShare` 后：

```bash
./scripts/copy-wechat-sdk-to-appshare.sh   # 首次或更新 SDK
./scripts/prepare-offline-hello-archive.sh
# Xcode Archive → install-ios-debug-ipa.sh
```

仅改 `.vue` / `js_sdk` 可 HBuilderX 热更新，**不必**重打基座。
