#### .appex（WidgetKit Extension）单独打包：

- 以模板工程形式（不是 framework）放在 SDK 的资源目录；
- 用户（宿主 App）在集成时：
  - 将 `.appex` 拖入自己的工程；
  - 修改 appex 的 Bundle ID；
  - 勾选 “Embed App Extensions”；
  - 共享同一个 `Attributes.swift` 模型（从 SDK 导入）。

5. 不能与主SDK 并存

**Widget Extension 引用了主 SDK，而主 SDK 又 transitively 引入了 React Native 库。**
 → React Native 底层的某些库使用了 `UIApplication.sharedApplication`。

 **App Extension（包括 WidgetKit、Share Extension、Notification Service Extension 等）** 中，
 Apple 明确禁止访问 `UIApplication.sharedApplication`，因为：

> Extension 无法控制整个 App 生命周期，只能运行在宿主环境中。

> 灵动岛 / Live Activities 必须作为 `.appex` 独立签名与嵌入，
> SDK 只封装逻辑 + 提供 appex 模板，宿主 App 集成时嵌入该 appex 即可。

 一、先说结论：

> ⚠️ **灵动岛相关内容（WidgetKit / ActivityKit appex）不能直接打包进单一 SDK Framework 里分发**。

- `.appex`（扩展）在 iOS 系统中 **必须独立签名**；
- `Framework` 无法携带已签名的 `.appex`；
- `.appex` 需要有独立的 `Bundle Identifier`、`Info.plist`、`NSExtension` 配置；
- 系统在运行时只识别主 App 中的 **PlugIns 目录下的 appex**。

 二、为什么不能直接放进 SDK？

| 机制                       | 限制                                                         |
| -------------------------- | ------------------------------------------------------------ |
| **Framework 机制**         | 用于共享代码，不允许嵌入签名的 appex 资源                    |
| **App Extension (.appex)** | 必须位于主 App 的 `PlugIns/` 中，由主 App 一起签名           |
| **签名规则**               | appex 的 bundle id 必须以主 App 的 bundle id 为前缀          |
| **SDK 分发逻辑**           | SDK 通常是被多个 App 使用，bundle id 不确定，因此无法提前签名 appex |

### 图片 下载方法

https://blog.csdn.net/SOHU_TECH/article/details/149819894
网络图片的展示

1. 技术限制
    因为Live Activity内部禁用网络，所以你在Target中不要以任何的希望进行网络请求，比如AsyncImage或者URLSession都是无效的，如果你希望展示网络图片,一种是在创建的时候提前在主项目中下载好，然后存储到一个公共区域(App Group)这个也需要你的开发者账号进行配置，然后在Target项目中需要展示的时候去公共区域进行加载。

2. 解决方案
    该方法写在主项目中的管理实时活动的延展类中，下载图片方法：

  APP下载-->缓存-->放入groupid,并userdefault保存-->传路径到灵动岛
  灵动岛-->接收路径-->路径去groupid获取-->展示

  灵动岛-->接收http地址(服务端推送)-->groupid 本地userdefault获取-->展示

```objc
//并发下载图片任务
- (void)downloadImages:(NSDictionary<NSNumber *, NSString *> *)urlMap
            completion:(void (^)(NSDictionary<NSNumber *, NSString *> *result))completion {
    
    if (urlMap.count == 0) {
        completion(@{});
        return;
    }
    
    dispatch_group_t group = dispatch_group_create();
    dispatch_queue_t lockQueue = dispatch_queue_create("image.download.lock", DISPATCH_QUEUE_SERIAL);
    
    NSMutableDictionary<NSNumber *, NSString *> *result = [NSMutableDictionary dictionary];
    
    [urlMap enumerateKeysAndObjectsUsingBlock:^(NSNumber *key,
                                                NSString *urlString,
                                                BOOL *stop) {
        
        dispatch_group_enter(group);
        
        [self downloadImage:urlString completion:^(NSString * _Nullable imagePath) {
            
            if (imagePath) {
                dispatch_sync(lockQueue, ^{
                    result[key] = imagePath;
                });
            }
            
            dispatch_group_leave(group);
        }];
    }];
    
    dispatch_group_notify(group, dispatch_get_main_queue(), ^{
        NSDictionary *resultCopy = result.copy;
        MDDebugLog(@"图片就绪（缓存或下载）请求数=%lu 成功数=%lu keys=%@", (unsigned long)urlMap.count, (unsigned long)resultCopy.count, resultCopy.allKeys);
        completion(resultCopy);
    });
}

/// 将图片 URL -> 本地路径 写入 App Group UserDefaults，供灵动岛查表
- (void)la_saveImagePathToGroup:(NSString *)localPath forKey:(NSString *)urlKey groupID:(NSString *)groupID {
    if (!localPath.length || !urlKey.length || !groupID.length) { return; }
    NSUserDefaults *suite = [[NSUserDefaults alloc] initWithSuiteName:groupID];
    [suite setObject:localPath forKey:urlKey];
    [suite synchronize];
    MDDebugLog(@"[LA_IMAGE_SAVE] groupID=%@ | key=%@ | value=%@", groupID, urlKey, localPath);
}

//下载图片
- (void)downloadImage:(NSString *)urlString
           completion:(void (^)(NSString * _Nullable imagePath))completion {
    
    if (urlString.length == 0) {
        dispatch_async(dispatch_get_main_queue(), ^{
            completion(nil);
        });
        return;
    }

    // 使用 group.包名，适用多 APP
    NSString *groupID = [NSString stringWithFormat:@"group.%@", [DeviceUtil getPackageName]];
    NSURL *containerURL =
    [[NSFileManager defaultManager]
     containerURLForSecurityApplicationGroupIdentifier:groupID];

    if (!containerURL) {
        dispatch_async(dispatch_get_main_queue(), ^{
            completion(nil);
        });
        return;
    }

    // 固定缓存目录
    NSURL *imageDir = [containerURL URLByAppendingPathComponent:@"live_activity"
                                                    isDirectory:YES];

    NSFileManager *fm = [NSFileManager defaultManager];
    if (![fm fileExistsAtPath:imageDir.path]) {
        [fm createDirectoryAtURL:imageDir
     withIntermediateDirectories:YES
                      attributes:nil
                           error:nil];
    }

    // 缓存文件路径（提前算好）
    NSString *fileName = [NSString stringWithFormat:@"%@.png", [self md5:urlString]];
    NSURL *fileURL = [imageDir URLByAppendingPathComponent:fileName];

    // 1️⃣ 先命中缓存（关键）
    if ([fm fileExistsAtPath:fileURL.path]) {
        MDDebugLog(@"✅ 命中缓存: %@", fileURL.path);
        [self la_saveImagePathToGroup:fileURL.path forKey:urlString groupID:groupID];
        dispatch_async(dispatch_get_main_queue(), ^{
            completion(fileURL.path);
        });
        return;
    }

    // 2️⃣ 缓存未命中，开始下载
    NSURL *url = [NSURL URLWithString:urlString];
    if (!url) {
        dispatch_async(dispatch_get_main_queue(), ^{
            completion(nil);
        });
        return;
    }
    NSURLSessionDataTask *task =
    [[NSURLSession sharedSession] dataTaskWithURL:url
                                completionHandler:^(NSData * _Nullable data,
                                                    NSURLResponse * _Nullable response,
                                                    NSError * _Nullable error) {

        if (!data || error) {
            MDDebugLog(@"⛔️ 图片下载失败: %@", error);
            dispatch_async(dispatch_get_main_queue(), ^{
                completion(nil);
            });
            return;
        }

        BOOL success = [data writeToURL:fileURL atomically:YES];
        if (success) {
            MDDebugLog(@"✅ 图片缓存成功");
            [self la_saveImagePathToGroup:fileURL.path forKey:urlString groupID:groupID];
        } else {
            MDDebugLog( @"⛔️ 图片写入失败");
        }

        dispatch_async(dispatch_get_main_queue(), ^{
            completion(success ? fileURL.path : nil);
        });
    }];

    [task resume];
}
- (NSString *)md5:(NSString *)string {
    const char *cStr = [string UTF8String];
    unsigned char digest[CC_MD5_DIGEST_LENGTH];
    CC_MD5(cStr, (CC_LONG)strlen(cStr), digest);
    
    NSMutableString *output = [NSMutableString stringWithCapacity:CC_MD5_DIGEST_LENGTH * 2];
    for (int i = 0; i < CC_MD5_DIGEST_LENGTH; i++) {
        [output appendFormat:@"%02x", digest[i]];
    }
    return output;
}
```
