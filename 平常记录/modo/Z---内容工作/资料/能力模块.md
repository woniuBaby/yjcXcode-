🔧 模块功能详解

1. **基础层模块**

```
Basic        - 基础工具类、通用组件
config       - 应用配置管理
Log          - 日志系统
Network      - 网络请求封装
Storage      - 数据存储（本地/缓存）
Permission   - 权限管理
```

2. **核心业务模块**

```
Login        - 用户登录认证
AccountVerification - 账号验证
Pay          - 支付系统
game         - 游戏核心模块
Player       - 播放器（音视频）
MusicEngine  - 音乐引擎
```

3. **音视频通信模块**

```
Media        - 媒体处理（图片/视频）
TRTC         - 腾讯实时音视频（Tencent RTC）
RealtimeVoiceTrans - 实时语音转换
VoiceTranslation - 语音翻译
```

4. **功能服务模块**

```
Push         - 消息推送
Share        - 社交分享
Map          - 地图定位
DeepLink     - 深度链接
FileFuc      - 文件功能
FaceRecognition - 人脸识别
```

5. **商业化模块**

```
Reward_video - 激励视频广告
Max          - 可能指 AppLovin MAX 广告平台
BIReport     - 商业智能报表
Report       - 数据上报
```

6. **架构相关**

```
Plugin       - 插件系统
Event        - 事件总线
entry        - 应用入口
```
